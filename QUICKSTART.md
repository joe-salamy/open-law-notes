# Quickstart

> **First time here?** Read the full README setup guide before using this file.
> Come back here once setup is complete.

---

These are the only three commands you need to run OpenLawNotes each time.

**Before you start:** find the `cd` command you used in Step 3 of the README — the one that navigated to the project folder. Copy it and paste it in place of the placeholder below. Save this file so you always have it ready.

---

## Windows (Command Prompt)

```
cd C:\path\to\open-law-notes
```

_(replace this with your actual project path)_

```
venv\Scripts\activate
```

```
python main.py
```

---

## Mac (Terminal)

```
cd /path/to/open-law-notes
```

_(replace this with your actual project path)_

```
source venv/bin/activate
```

```
python main.py
```

---

> **Reading-only mode** (skip audio, only process readings):
>
> ```
> python main.py --read-only
> ```
